godot-suite

Curated Godot 4.x indie game dev suite for Claude Code.

Install via this marketplace (catalog served here; plugin hosted on GitHub):

claude plugin marketplace add https://godot-suite.llm4agents.com/marketplace.json
claude plugin install godot-suite@godot-suite

Or straight from GitHub (clones the whole repo):

claude plugin marketplace add llmforagents/godot-suite
claude plugin install godot-suite@godot-suite
